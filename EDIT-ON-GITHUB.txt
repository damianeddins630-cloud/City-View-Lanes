────────────────────────────────────────
HOW TO CHANGE A FILE ON GITHUB
────────────────────────────────────────

1) Open the repo link above (log in as damianeddins630)
2) Click the folder / file you want
3) Click the pencil icon (Edit this file)
4) Make your change
5) Scroll down → Commit changes → Commit directly to the main branch
6) If Vercel is connected to this GitHub repo, the site updates by itself
   (see “Connect Vercel to GitHub” below)

  src/components/CareersClient.tsx

HEADER / FOOTER / SITE NAME / PHONE
  src/components/Header.tsx
  src/components/Footer.tsx
  src/lib/site.ts
  → phone, address, tagline, nav links, Pro Shop URL

PHOTOS (in the project)
  public/images/
  → replace a file with the SAME name to swap a photo
  → or upload a new image and change the path in the page file

DEFAULT LEAGUES / ROLES (seed data)
  data/store.json
  → only the starting data; live site saves go to Vercel Blob
  → for live leagues/hours/users use Admin on the website

ADMIN / PERMISSIONS LABELS
  src/lib/site.ts          → permission checkboxes
  src/lib/roles.ts         → Website Owner vs Admin rules

5) Save
