CityView Lanes — make saves stick on Vercel
==========================================

IMPORTANT: Always upload this zip into your EXISTING Vercel project.
Do NOT click "Create New Project" every time, or Blob settings get lost.

Step-by-step (about 2 minutes)
------------------------------
1. Open https://vercel.com and open the project that runs your live CityView site.

2. Click Storage → Create Database / Blob
   - Name it anything (e.g. cityview-data)
   - Private or Public both work with this build
   - Connect it to THIS project (Production)

3. Confirm env vars exist:
   Settings → Environment Variables
   You should see at least one of:
   - BLOB_STORE_ID
   - BLOB_READ_WRITE_TOKEN

4. Deployments → open the latest deployment → Redeploy
   (Required after connecting Blob. Upload alone is not enough if Blob was just added.)

5. Open your live site → Login as masteradmin → Admin
   - Banner should say Saves are on / mode blob (not memory)
   - Click "Test save now" → must say PASS

6. Edit Hours → Save → refresh /hours → times should stay.

If mode is still "memory"
-------------------------
You are looking at a different Vercel project than the one with Blob,
or you have not Redeployed after connecting Blob.
